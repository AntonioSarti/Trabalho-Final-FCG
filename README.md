# Trabalho-Final-FCG
O trabalho desenvolvido da disciplina de Fundamentos de Computação Gráfica (FCG) - INF01047 - UFRGS, 
trata-se de um jogo inspirado em Vampire Survivors, com mecânicas simples, como ataque automático e 
hordas de inimigos. É um jogo com ação contínua, onde o jogador se vê constantemente desafiado a 
evoluir suas habilidades e sobreviver contra inimigos cada vez mais fortes.
